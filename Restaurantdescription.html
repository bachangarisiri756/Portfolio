About Project

Tasty Bites Restaurant is a simple restaurant website designed to display food items and menu details for customers. The website provides information about delicious dishes and their prices in an attractive layout.

Description

This website is created using HTML and CSS. It includes a header section with a restaurant background image and a menu section displaying different food items like Chicken Biryani, Veg Biryani, and Paneer Butter Masala. Each menu item shows an image, description, and price.

Menu Items

  • Chicken Biryani – ₹180
  • Veg Biryani – ₹150
  • Paneer Butter Masala – ₹170

Features of the Website

  • Restaurant background header
  • Food images for menu items
  • Menu description with prices
  • Simple and attractive layout
  • Organized content using sections

Technologies Used

  • HTML – Structure of the webpage
  • CSS – Styling and layout design
  • Images – To display food items
  • Float layout – For menu alignment

Purpose of the Website

The main purpose of this restaurant website is to display the restaurant menu in a simple and attractive way. It helps customers view available dishes and their prices before ordering food.
