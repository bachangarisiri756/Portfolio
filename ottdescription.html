OTT Platform Project

This project is a simple OTT Platform system developed using Java and data structures concepts. It allows users to explore movies and manage their watch lists efficiently.

Description

The system stores movie details such as Movie ID, Movie Name, Genre Code, Ratings, and Views. Users can check available movies, search movies, and view trending movies.

Features

  • Display available movies
  • Search movies by ID, name, or genre
  • Trending movies list
  • Watch history management
  • Watch list using stack
  • Sorting movies by ratings, views, and name

Technologies Used

  • Java Programming
  • Arrays
  • Stack
  • Queue
  • Searching and Sorting

Purpose of Project

The main purpose of this project is to demonstrate the use of data structures like stack, queue, and arrays in a real-world application such as an OTT movie platform.
